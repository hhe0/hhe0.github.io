结构体

发表于 2019-07-31   |   更新于 2019-07-31 | | 阅读次数
字数统计: | 阅读时长 ≈

结构体是将零个或者多个任意类型的命名变量组合在一起的聚合数据类型。

结构体的定义

  • 结构体的成员变量通常一行写一个，变量的名称在类型的前面，但是相同类型的连续成员变量可以写在一行（不推荐）；
  • 成员变量的顺序对于结构体同一性很重要。两个结构体的成员顺序不同也为不同的结构体；
  • 如果一个结构体的成员变量名称是首字母大写的，那么这个变量就是可导出的。一个结构体可以同时包含可导出和不可导出的成员变量；
  • 结构体类型s不可以拥有一个同类型s的成员，但是可以拥有一个结构体指针类型*s的成员；
  • 结构体的零值由结构体成员的零值组成。

匿名成员

  • go允许我们定义不带名称的结构体成员，只需要指定类型即可；这种结构体成员称作匿名成员，该结构体成员的类型必须是一个命名类型或者指向命名类型的指针。

    定义

访问与赋值

  • 通过定义匿名成员，我们可以直接访问到匿名成员的成员和相关的方法；
  • 不能在一个结构体中定义两个相同类型的匿名成员，否则会引起冲突。

初始化

  • 没有什么快捷方式来初始化具有匿名成员的结构体，必须遵循结构体类型的定义来初始化。

结构体初始化

  • 可以使用不指定成员或者指定成员两种方式来初始化结构体，建议使用指定成员这种方式；
  • 如果一个成员没有被初始化，则会得到该成员的零值。

作为函数的参数

  • 处于效率的考虑，大型的结构体通常都使用结构体指针的方式直接传递给函数或者从函数中返回；
  • 这种方式在函数需要修改结构体内容是必需的，Go是使用按值调用的语言，调用的函数接收到的是实参的一个副本，而不是实参的引用。

结构体的比较

  • 如果结构体的所有成员变量都可以比较，那么这个结构体就是可比较的；
  • 比较使用==或者!=;
  • 可比较的结构体可以作为map的键类型。

【Python编程快速上手(四)】

发表于 2019-01-14   |   更新于 2019-05-07 | 分类于 Python读书笔记系列 | | 阅读次数

使用列表来处理多个数据。

列表数据类型

用下标取得列表中的单个值

  • 下标值不能超过列表中元素的个数；
  • 下标只能是整数，不能是浮点数。

负数下标

  • -1表示最后一个元素，以此类推。

利用切片取得子列表

用len()取得列表的长度

用下标改变列表的值

列表连接和列表复制

用del语句从列表中删除值

  • 在实践中，你几乎永远不需要删除简单变量。del语句几乎总是用于删除列表中的值。

使用列表

  • 避免重复，便于管理。

列表用于循环

in和not in操作符

多重赋值技巧

  • 在一行代码中，用列表中的值为多个变量赋值；
  • 变量的数目与列表的长度必须严格相等。

增强的赋值操作

  • +=、-=、*=、/=、%=，其中+=和’-=’也可以连接/复制字符串或者列表。

方法

用index()方法在列表中查找值

  • 如果有多个值，则返回第一个下标。

用append()和insert()方法在列表中添加值

  • append()方法将元素添加到列表的末尾，insert()方法将元素添加到列表的首部。这两个方法都没有返回值。

用remove()方法从列表中删除值

  • 如果该值在列表中出现多次，则只删除第一次出现的值；
  • 删除列表中不存在的值，会出现ValueError错误；
  • 如果知道想要删除的值在列表中的下标，del语句就很好用。如果知道想要从
    列表中删除的值，remove()方法就很好用

用sort()方法将列表中的值排序

  • sort()方法没有返回值；
  • 不能对既有数字又有字符串值的列表进行排序；
  • sort()方法对字符串排序时，使用『ASCII字符顺序』，而不是实际的字典顺序。因此，会导致小写字母在大写字母在后，需要改变顺序的话，可以传递参数key=str.lower

类似列表的类型：字符串和元祖

  • 字符串是单个文本字符的列表；
  • 对列表的许多操作，也可以作用于字符串：按下标取值、切片、用于 for 循环、用于 len()，以及用于 in 和 not in 操作符

可变和不可变数据类型

  • 列表是可变数据类型，字符串是不可变数据类型；
  • 更新一个字符串是通过切片和连接完成的。

元组数据类型

  • 元组输入时用圆括号()，而不是方括号[]；
  • 元组与字符串一样属于不可变类型。

用list()和tuple()函数来转换类型

引用

  • 变量包含对列表值的引用，而不是列表值本身；
  • 对于字符串和整数值，变量就包含了字符串或整数值。

传递引用

  • 理解参数如何传递给函数，引用就特别重要。

copy()模块的copy()和deepcopy()函数

  • copy.copy()用来复制列表或字典这样的可变值，而不只是复制引用；
  • 如果要复制的列表中包含了列表，那就使用copy.deepcopy()函数来代替，deepcopy()函数将同时复制它们内部的列表。

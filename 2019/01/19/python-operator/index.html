Python中的运算符

发表于 2019-01-19   |   更新于 2019-05-07 | 分类于 python随笔 | | 阅读次数

Python中的运算符与其他类C语言的运算符有一些区别，注意区分掌握。

算术运算符

运算符 描述 说明
+ 加
- 减
* 乘
/ 除 特别注意/与//的区别
% 取模
** 幂 python中很有特色的运算符
// 整除 向下取整

比较运算符

运算符 描述 说明
== 等于
!= 不等于 注意没有<>运算符
> 大于
< 小于
>= 大于等于
<= 小于等于

赋值运算符

运算符 描述 说明
= 简单的赋值运算符
+= 加法赋值运算符
-= 减法赋值运算符
*= 乘法赋值运算符
/= 除法赋值运算符
%= 取模赋值运算符
**= 幂赋值运算符
//= 取整除赋值运算符

位运算符

运算符 描述 说明
& 按位与
\ 按位或
^ 按位异或
~ 按位取反
<< 左移运算符
>> 右移运算符

逻辑运算符

运算符 描述 说明
and 布尔与
or 布尔或
not 布尔非

成员运算符

  • 适用于字符串、列表和元组。
运算符 描述 说明
in 在指定的序列中返回True，否则返回False
not in 不在指定的序列中返回True，否则返回False

身份运算符

  • 用于比较两个对象的存储单元。
运算符 描述 说明
is 判断两个标识符是不是引用自一个对象
is not 判断两个标识符是不是引用自不同对象

三目运算符

运算符优先级

运算符 描述 说明
** 指数
~、+、- 按位翻转, 一元加号和减号
*、/、%、// 乘、除、取模和取整除
+、- 加法减法
>>、<< 右移，左移运算符
& 位 ‘AND’
^、\ 位运算符
<=、<、>、>= 比较运算符
==、！= 等于运算符
=、%=、/=、/=、-=、+=、*=、**= 赋值运算符
is、is not 身份运算符
in、not in 成员运算符
and、or、not 逻辑运算符
